HTML Structure

All of the information within the main content is nested within multiple section with tag of "section". Here is the general structure. All style.css are put in head tag, put all js files(In addition to jquery.js) before </body> tag.

CSS Files and Structure

There are 9 main CSS files in this theme.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

Open the css folder from this location 'assets/css', you'll find all css files.

  • main.css - theme common styles
  • bootstrap.min.css - Twitter bootstrap styles
  • font-awesome.min.css - Font Awesome stylesheet
  • owl.carousel.css - App screen, tesimonial, press and gallery blog carousel main style
  • owl.theme.css - App screen, tesimonial, press and gallery blog carousel themes style
  • magnific-popup.css - magnific-popup styles /
  • spinner.css - Preloader styles /
  • animate.min.css - Animation Style

Logo & Menu Settings

Change your logo text and link navbar markup:

    <!-- logo -->
    <a class="navbar-brand text-logo" href="index.html"> appee </a>

If you want to use image logo use this code and remove text logo code:

    <!-- logo -->
    <a class="navbar-brand logo" href="index.html"><img src="assets/img/logo.png" alt="logo"> </a>

The Image logo size is limited max height as 25px.

MENU:

Edit menu here:

 

Each <li> tag  is a menu item, you could edit the text and link.

	<li><a data-scroll href="#home">HOME</a> </li>

If you want to add submenu in Menu item, just insert code  like this:

	<li class="dropdown"><a href="#" class="dropdown-toggle" data-toggle="dropdown" data-hover="dropdown" data-delay="10">MORE</a>
	<ul class="menulist dropdown-menu">
	<li><a href="../Windows/index-background-video.html">Windows Version</a></li>
	</ul>
	</li>
